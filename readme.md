## 1.2 订单报名信息更新接口
    .post("/Registration/update")
    传递的参数和新增的时候一致，非null会自动覆盖，必须传递id


## 1.3 订单报名信息获取接口
### 1.3.1 根据报名id获取单个信息
    get("/Registration/{id}"
### 1.3.2 根据weixinId获取全部报名信息
    .get("/Registration/getByWeixinId")
